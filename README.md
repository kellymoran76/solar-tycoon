# Solar Idle ☀️⚡

A lightweight idle/clicker solar game built with vanilla HTML/CSS/JS — no build tools.

## Features
- Click to generate energy (kWh)
- Solar production with day/night curve + occasional cloud events
- Buy Panels, Inverters (sell cap), Trackers (sun bonus), Batteries (storage)
- Upgrades, Achievements
- Autosave + Offline progress
- Import/Export save
